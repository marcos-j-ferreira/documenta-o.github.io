Cenário 1 (Cadastro)- Cadastro bem-sucedido:

Verifica se o Cadastro funciona com credenciais válidas.

Tela de cadrastro: Cadastro


O teste coloca credenciais válidas e verifica o funcionamento do código./p>

Repositorio do teste

Cenário 2 (Cadastro) - Campos obrigatórios:

Verifica se os campos est'ao sendo obrigatórios, antes de passar

Tela de cadrastro: Compos


Clicar no botão para entrar sem colocar os dados necessario, vefificando se o sistema pede dados obrigatórios/p>

Repositorio do teste

Cenário 3 (Cadastro) - Validação do formato do E-mail:

Verifica se o sistema pede um formato do email certo

Tela de Login: Login


O teste coloca um formato de e-mail inválido, tenta fazer o login e espera-se que não seja realizado/p>

Repositorio do teste

Cenário 4 (Login) - Login bem-sucedido:

Testa se o sistema de login está funcionando como esperado, inserido dados validos

Tela de LOGIN LOGIN


O teste coloca credenciais válidas e verifica o funcionamento do código, ao fazer o login./p>

Repositorio do teste

Cenário 5 (Login) - Login com credenciais erradas:

Verifica se o Cadastro não funciona com credenciais inválidas.

Tela de Login: Cadastro


Inserido dados inválida e verifica o funcionamento do código./p>

espera-se que o login não sej feito

Repositorio do teste

Cenário 6 (Login) - Login com conta inexistente:

Verifica se o Login funciona com dados inexistente.

Tela de Login: Cadastro


São inseridos dados inexistentes no login, e não deve ser feito o login, garantindo a segurança do sistema
